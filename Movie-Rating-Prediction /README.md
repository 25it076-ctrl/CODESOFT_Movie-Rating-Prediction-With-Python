# Movie Rating Prediction with Python

## Project Overview

The Movie Rating Prediction project is a machine learning application that predicts movie ratings based on features such as genre, director, and actors. The model analyzes historical movie data and estimates the rating a movie is likely to receive.

## Objective

The objective of this project is to build a regression model that can accurately predict movie ratings using machine learning techniques and historical movie information.

## Dataset

The dataset contains information about movies with the following features:

* Genre
* Director
* Actor
* Rating

The categorical features are converted into numerical values using Label Encoding before training the model.

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Machine Learning Algorithm

* Random Forest Regressor

## Project Workflow

1. Load the movie dataset.
2. Preprocess the data using Label Encoding.
3. Split the dataset into training and testing sets.
4. Train the Random Forest Regressor model.
5. Predict movie ratings.
6. Evaluate the model using Mean Absolute Error (MAE).

## Results

The model successfully predicted movie ratings based on movie characteristics and achieved satisfactory performance on the test dataset.

## Learning Outcomes

* Understanding regression-based machine learning models.
* Data preprocessing and feature encoding.
* Model training and evaluation.
* Predictive analysis using Random Forest Regression.
